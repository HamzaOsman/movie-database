Web application similar to the Internet Movie Database. The application maintains a list of movie information. The website supports regular
users, who are capable of browsing all of the information on the site and adding movie reviews. It also contributing users, who can add new people/movies and edit existing records. The site will also offer movie recommendations based on a userâ€™s past reviews.

To initialise and run the server:
1) Navigate to the directory containing 'server.js' and open Powershell. Within Powershell type 'npm install' to install all the required modules (express,pug,express-session)

2) Type 'node server.js' to run the website.

3) Finally, enter 'http://localhost:3000/' into a web browser to access the website.
